Get to know me!

您好！我是 莊昀展 Gary，一位具備機械工程嚴謹邏輯與全端開發實力的跨領域開發者。

在相關作品中將會看到我運用 React 與 Tailwind CSS 打造兼顧畫面質感與實際體驗的前端介面，並能獨立建置基於微軟 C# (.NET Minimal API) 與雲端 PostgreSQL (Neon) 資料庫的穩定後端系統。

身為非資訊本科背景學生，我擁有極強的自學熱忱與運用 AI 協作開發的能力，能貫穿痛點分析、產品企劃到技術部署的完整環節，也對 UI/UX 設計具備熱情，渴望在具挑戰性的相應環境中，創造實質的產品影響力。
